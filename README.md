# NelxRatedCurrency

WoW addon (Midnight Season 1 · Interface 120001) that tracks PvP currencies across all your characters — honor, conquest, and more — with tooltip integration and a full overview panel.

## Features

- **Currency Tracking** — captures Honor and Conquest per character automatically on login and currency updates
- **Tooltip Extension** — hover any PvP currency icon or item to see amounts for all your characters, colored by class
- **Overview Panel** — `/nrc` opens a sortable table of all characters × currencies
- **Column Sorting** — click any column header to sort by character name or currency amount
- **Minimap Button** — left-click to toggle panel, right-click for settings; draggable, position saved
- **Settings Panel** — hide individual characters from tooltips and the overview, or disable tooltip extension entirely

## Usage

| Action | How |
|--------|-----|
| Open overview | `/nrc` or left-click minimap button |
| Open settings | Right-click minimap button |
| Hide a character | Settings → Characters → uncheck |
| Disable tooltip | Settings → Disable tooltip extension |

## Installation

Copy the `NelxRatedCurrency` folder to:

```
WoW/_retail_/Interface/AddOns/NelxRatedCurrency
```

Then reload your UI: `/reload`
